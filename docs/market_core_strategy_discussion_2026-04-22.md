# 市场核心策略讨论纪要（2026-04-22）

## 1. 讨论背景

当前“市场观察日报”里的选股/核心标的逻辑，用户明确表示不满意。主要问题不是排序细节，而是策略目标本身偏了：

- 现在选出来的“核心标的”并不是真正的市场核心。
- 当前 `dragon / center / follow / watchlist` 这套角色框架，对当前市场风格的解释力有限。
- 现在市场整体风格偏趋势，不是强情绪、强连板、强妖股主导。
- 因此，策略核心目标应从“角色打分+配额凑池”转向“识别主线中的题材核心”。

本纪要只归纳本轮讨论结论，不代表最终定稿方案。

## 2. 市场风格前提

当前讨论默认以下市场假设：

- 市场环境整体不差，存在赚钱效应。
- 市场不太鼓励纯连板妖股式炒作。
- 趋势型主线、容量型核心更重要。
- 情绪标的仍然有参考价值，但不再是核心判断主轴。

这意味着后续系统识别的优先级，应从“情绪龙头”切换到“题材核心”。

## 3. 当前策略的核心问题

本轮讨论中，已明确认为当前策略存在以下问题：

1. `dragon / center / follow` 这套角色逻辑意义不大，至少不适合作为核心标的识别主框架。
2. 观察池 Top20 的配额式筛选，容易把非核心杂毛补进去。
3. 当前逻辑更像在做“结构化观察池”，而不是识别真正的市场核心。
4. 真正的核心，不能只靠单日涨跌或角色模板判断，而要看趋势、容量、地位、唯一性和板块聚集度。

## 4. 本轮讨论后的目标重定义

目标不再是“给全市场股票做角色分层后输出 Top20”，而是回答以下问题：

- 现在市场最热、最强的方向在哪里？
- 这个方向里，谁是题材核心？
- 谁是容量核心？
- 谁是情绪/弹性标的？
- 这些角色是否集中在同一板块/方向上共振？

用户给出的核心判断逻辑是：

> 当环境、板块热度、核心标的、容量标的、情绪标的都集中到一个板块时，才更像确定性较高的赚钱效应方向。

## 5. 题材核心的定义

本轮讨论中，已明确：

- 后续识别要以“题材核心”为主。
- 情绪核心不是主轴。
- 容量核心也重要，但应服务于题材核心判断，而不是替代题材核心。

用户给出的代表性例子：

- AI 硬件 / CPO 的 `中际旭创`
- 新能源阶段的 `阳光电源`
- 消费白酒中的 `茅台`

这些票的共同点不是连板高度，而是：

- 市场共识强
- 容量承接强
- 趋势持续强
- 能代表板块或产业方向

## 6. 题材核心识别的优先级

本轮讨论明确的优先级如下：

1. 市场共识
2. 容量趋势
3. 产业逻辑

但需要强调：

- 这里的“产业逻辑”不准备主要靠文本理解或新闻语义来识别。
- 讨论中已经倾向于把“逻辑硬度”降为数据代理信号，不用主观文本判断。

因此，后续系统更像在做：

- 有共识的趋势核心
- 有容量承接的板块锚点

而不是做“新闻语义龙头识别”。

## 7. 个股层面已达成的口径

### 7.1 趋势维度

后续要补充并重点使用：

- `5日涨幅`
- `10日涨幅`
- `20日涨幅`

使用原则：

- 既看绝对涨幅
- 也看板块内相对地位
- 以板块内相对排名为主
- 绝对涨幅为辅

原因：

- 核心判断的关键不是“全市场涨了多少”，而是“在这条主线里是不是最强、最持续的那个”。

### 7.2 市场共识

用户明确表示：

- 新闻源当前不够稳定，不能把新闻提及作为主判断依据。
- 市场共识应做成综合信号。
- 但新闻相关权重应放低。

因此后续“市场共识”更适合用结构化数据做低权重综合判断，而不是靠文本主导。

### 7.3 热度

热度维度明确了两个原则：

- 热度权重可以降低
- 直接使用现有表结构里已有的结构化热度数据

也就是说：

- 热度不是决定它是不是核心
- 只是辅助确认“市场有没有在看它”

### 7.4 板块地位

用户明确认为，板块内地位要重点看三个因素：

- 绝对高度
- 唯一性
- 成交量 / 成交额承接

其中：

- “绝对高度”不应只理解为连板高度
- 在趋势市里，更应扩展为阶段性高度、5/10/20 日趋势高度

### 7.5 唯一性

唯一性明确要求优先用纯数据判断，不依赖主观文本解释。

本轮讨论中确定的主要口径：

1. 同板块内，`5日 / 10日 / 20日` 趋势显著领先
2. 同板块内，同时具备 `高热度 + 高成交 + 高趋势`

不优先使用“市场反复提及它是代表”这种文本语义判断。

### 7.6 容量 / 承接

本轮讨论已确定：

- 看绝对成交额
- 看板块内成交额排名
- 使用方式为：
  - 板块内排名为主
  - 绝对成交额做门槛

原因：

- 只看绝对成交额会天然偏向大票
- 只看排名又可能让小板块里的弱承接票误判成核心

### 7.7 “逻辑硬”如何处理

原本“逻辑硬”容易走向主观文本解释，但本轮讨论明确收口为数据代理信号。

优先级已定为：

1. 成交量 / 容量承接
2. 趋势持续性

主要参考：

- `5日涨幅`
- `10日涨幅`
- `20日涨幅`
- 成交额
- 板块内成交额排名

因此，后续不一定需要单独设计一个强主观的 `logic_score`，更可能把这部分合并到：

- `trend_score`
- `liquidity_score`

里去表达。

## 8. 板块层面的讨论结论

本轮讨论中，用户指出一个关键问题：

- 不能只盯个股
- 也要看板块
- 而且要看板块的 `5日 / 10日 / 20日涨幅`
- 还要看板块的成交资金、成交量

这意味着后续板块判断不应只看：

- 当日板块涨幅
- 当前 `board_score`

而要补充更完整的板块趋势和资金维度。

## 9. 当前代码/数据层的现状认知

本轮讨论中，已初步确认以下现状：

### 9.1 个股数据

- A 股个股日线当前已切到 `BaoStock` 为默认来源。
- 这部分后续补 `5/10/20 日涨幅` 是可行的。

### 9.2 板块数据

当前板块数据来源不是单一来源，而是混合口径：

1. 原始行业/概念板块快照
   - 主要通过 `AkShare`
   - 优先走 THS / EM 相关接口

2. 日跑里额外构建的证监会行业板块
   - 先用 `BaoStock` 拉行业归属
   - 再结合个股日线数据聚合生成 `industry_csrc`

初步结论：

- 板块数据当前不是完全由 `BaoStock` 提供
- 也不是完全由 `AkShare` 提供
- 行业板块的一部分已经能通过 `BaoStock + 个股行情聚合` 得到

### 9.3 板块 5/10/20 日涨幅现状

当前没有显式的：

- `board_pct_chg_5d`
- `board_pct_chg_10d`
- `board_pct_chg_20d`

当前仅有一个 `continuity_score`，但这不是等价替代。

### 9.4 板块资金现状

当前板块层已有部分资金信息：

- `total_amount`
- `top_amount`

但它们目前只是被用于现有的板块评分子项，没有被独立建模成新的板块强度判断框架。

## 10. 对 BaoStock 与 AkShare 的阶段性认知

本轮讨论只形成了一个阶段性判断：

- `BaoStock` 可以覆盖一部分基础行情与行业归属数据
- 但不能简单认为 `AkShare` 的所有来源都能被 `BaoStock` 完整替代

尤其是以下方向，大概率仍需要 `AkShare` 或其他来源支撑：

- 概念板块
- 热度榜
- 涨停池
- 某些专题榜单/行为榜单

这一部分还没有在本轮讨论中完全定稿，后续需要继续细化。

## 11. 本轮讨论后的初步框架雏形

基于目前达成的共识，后续新策略更像下面这套逻辑：

1. 看市场环境是否支持趋势主线
2. 看哪些板块在趋势和资金上更强
3. 在强板块中找：
   - 题材核心
   - 容量核心
   - 情绪 / 弹性标的
4. 再判断这些角色是否集中到同一方向形成共振

题材核心的候选判断，当前倾向于基于以下维度综合：

- `consensus_score`
- `trend_score`
- `uniqueness_score`
- `liquidity_score`
- `theme_core_score`

但这些字段和计算方式都还没有最终定稿。

## 12. 当前明确暂不做的事情

本轮讨论中，明确没有定稿的内容包括：

- 最终评分公式
- 最终字段命名
- 板块 5/10/20 日涨幅的具体计算口径
- 板块成交量的最终定义
- 是否保留原 Top20 观察池结构
- 新旧策略如何过渡
- 如何在页面上展示“题材核心 / 容量核心 / 情绪标的”

## 13. 下次继续时建议优先讨论的问题

下次继续时，建议从以下问题继续：

1. 板块 `5日 / 10日 / 20日涨幅` 怎么算最合适
2. 板块资金与量能维度到底保留哪些字段
3. 新的“题材核心”是否要独立于旧的 `observation_pool`
4. 新框架里是否还要保留“容量核心 / 情绪标的”两条平行输出
5. `BaoStock` 与 `AkShare` 的边界如何最终划分

## 14. 结论

这轮讨论的核心结论不是“调权重”，而是：

- 当前策略目标要改
- 核心识别逻辑要从“角色打分+配额筛选”转向“趋势市下的题材核心识别”
- 关键证据要更多依赖结构化数据：
  - `5日 / 10日 / 20日涨幅`
  - 成交额 / 板块内成交额排名
  - 板块趋势
  - 板块资金
  - 唯一性
  - 低权重热度确认
